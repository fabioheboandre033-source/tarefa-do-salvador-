introdução

o css é uma linguagem usada para estilizar paginas wed. Com ele, você controla a aparelho do site,cores, fontes, tamanho, é como dar vida ao HTML, que só cuida da estrutura.

Por que usar CSS?

Ele é essencial para separar o conteúdo (HTML) do visual, deixando o código mais organizado e fácil de manter. Imagine mudar a cor de todos os títulos do site com apenas uma linha de código!

Como funciona?

Tipos de CSS

  1. Externo: Arquivo .css separado (ex: style.css).
  2. Interno: Dentro da tag *style* no HTML
  3. Inline: Diretamente no elemento HTML (ex: *p style="color:red;"*).

Vantagens

  1. Responsividade:Faz o site funcionar em qualquer tela (PC, celular).
  2. Consistência: Define estilos globais para todo o site.
  3. Fácil de manter:Altera o visual sem mexer no HTML.

Exemplo prático

body { background-color: #f0f0f0;
font-family: Arial, sans-serif;
}

importancia do CSS no desenvolvimento web

  1. separação de conteudo e estilos

    O CSS permite separar a estrutura (HTML) do design, deixando o código mais organizado e fácil de mater.

  2. Consistência visual

    Com o CSS, você define estilos globais (cores, fontes) que se aplicam a todo o site garantindo uma identidade visual coesa.

  3. Responsividade

    CSS é essencial para criar layouts que funcionam em qualquer dispositivo (PC, tablet, celular).

  4. compatibilidade

    CSS é suportado por todos os navegadores modernos, garantindo que o design funcione bem em qualquer ambiente

  5. SEO Amigavél

    Ao separar conteúdo (HTML) do estilo (CSS), os motores de busca leem melhor o conteudo, ajudando no ranqueamento
